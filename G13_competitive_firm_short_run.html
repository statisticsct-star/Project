Key rules
Profit-maximising output: if the firm produces, it chooses q* where P = MR = MC on the upward MC branch.
Shutdown rule: if P < min AVC, the firm shuts down in the short run because it cannot cover variable cost.
Break-even rule: at P = min ATC, economic profit is zero.
Firm supply: the short-run supply curve is the MC curve above the shutdown point.
Fixed-cost insight: changing fixed cost shifts AFC and ATC, but not MC, AVC, or supply.
