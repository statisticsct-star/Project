Why the three stages matter
Stage I: TP rises and AP rises because MP is above AP.
Stage II: TP still rises, but at a diminishing rate. MP is positive and below AP.
Stage III: MP becomes negative, so extra input reduces TP.
Model used in explore mode
TP(L) = A · K · (a·x + b·x² − c·x³), where x = L / K.
AP = TP / L and MP = dTP / dL.
The slider for K shifts the stage boundaries because more fixed capacity lets the firm use more of the variable input efficiently.

How the case study is translated into production-theory language
The zero-fertiliser plot acts as the fixed-factor baseline. The case-study “total product” is therefore the extra grain above the no-N control. That isolates the product of the variable input (applied nitrogen) while leaving land, irrigation, machinery, genetics, and background soil N in the baseline.
